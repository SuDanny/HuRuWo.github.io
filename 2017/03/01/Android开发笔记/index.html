eclipse工程转android studio编码问题

1.把AS设置为GBK编码

File–>Settings–>File Encodings–>都改为GBK
2.把gbk转为utf8编码文件
下载EditPlus
选中所有文件全部由EditPlus打开
Document–>File Encoding–>Convert Encoding Multiple
全部选中，点OK–>选中UTF8（千万别选带Bom的）
然后复制粘贴

Android Studio 使用github管理项目

1.分享项目到github

VCS–>Import into Version Control–>Share Project on Github

2.更改写入git文件中（上传前准备，更新到本地库）

点击图标VCS↑ 或者项目上右键–>git–>Commit Directory

3.提交到远程库

项目上右键–>git–>Repository–>Push
