独立开发一款App（斩码师）--初步设想，需求分析，基本架构

Android
  • Android
  • Java

前言

想做一个属于个人的项目。看见了一款刷单词的软件《百词斩》觉得很不错。于是希望能够独立开发一款类似的软件，用来刷面试题（或者在闲暇时间里阅读专业知识）。

需求分析

需求

只是个人想有这么一款软件，主要面对即将毕业找工作的学生。

目标人群

毕业大学生

希望实现的功能

基本功能

1.能够选择题库，比如java/jsp/数据库/android等知识点。

2.有后台，用户能够登录保存自己的的刷题进度。

3.题目以选择题为主，可以记录用户的错题/收藏以及化掉的题目。

4.遗忘曲线的分析，既记录刷过的题来推荐复习的题目。

其他功能

1.社交功能，用户能进行pk

2.题目可以自行用户笔记/评论

3.用户可以向服务器提交自己的题库

4.待更新。
